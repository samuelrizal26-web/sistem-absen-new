app_icon_fg.png – FOREGROUND LAUNCHER (Android)

Aturan desain agar icon tidak gepeng & lega:

• Ukuran canvas: 1024 × 1024 (1:1)
• Logo di TENGAH
• Logo ±65–70% dari canvas (jangan penuh/ mepet pinggir)
• SISA AREA = TRANSPARAN (atau kosong)
• Jangan pakai logo mentah penuh canvas

Background adaptive icon sudah diatur di config: #0E4C5A (gelap, kontras).

Ganti file ini dengan versi yang sudah diedit (logo lebih kecil + napas),
lalu jalankan: flutter pub run flutter_launcher_icons
